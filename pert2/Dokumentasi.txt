 Dokumentasi Program Pertemuan 2

1. Alur Kerja Program
   - Program dimulai dari class `UjiBarang` di method `main`.
   - Objek `gitar` dibuat menggunakan constructor dengan 2 parameter (kode, nama). Harga dan stok masih default (0).
   - Method `ubahHarga(1500000)` dipanggil pada objek `gitar` untuk mengeset harga.
   - Method `tambahStok(5)` dipanggil pada objek `gitar` untuk menambah jumlah stok.
   - Objek `drum` dibuat menggunakan constructor dengan 4 parameter (kode, nama, harga, stok), sehingga data langsung lengkap saat inisialisasi.
   - Program mencetak header laporan.
   - Method `tampilInfo()` dipanggil untuk kedua objek (`gitar` dan `drum`) untuk menampilkan status terakhir atribut-atributnya ke layar.

2. Fungsi/Method yang Digunakan
   - `BarangMusik(...)`: Constructor overloading. Ada 3 variasi:
     - 2 parameter (kode, nama): Inisialisasi dasar.
     - 3 parameter (kode, nama, harga): Inisialisasi dengan harga.
     - 4 parameter (kode, nama, harga, stok): Inisialisasi lengkap.
   - `ubahHarga(double)`: Mengubah nilai atribut harga barang.
   - `tambahStok(int)`: Menambahkan jumlah stok barang yang ada (akumulasi).
   - `tampilInfo()`: Menampilkan detail barang (Kode, Nama, Harga, Stok) dengan format rapi.
